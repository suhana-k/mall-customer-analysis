# 🛍️ Mall Customer Analysis

## 📌 Description

Customer data analysis using Python with visualization and clustering techniques to identify spending patterns and customer segments.

---

## 🎯 Objectives

* Understand customer demographics
* Analyze income and spending behavior
* Visualize relationships between variables
* Perform customer segmentation

---

## 📊 Dataset

The dataset contains 200 mall customers with:

* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)

---

## 📈 Analysis Performed

* Bar chart (Gender distribution)
* Histogram (Age, Income, Spending Score)
* Scatter plots (Income vs Spending)
* Boxplot (Outliers detection)
* Correlation heatmap
* K-Means clustering (Customer segmentation)

---

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🚀 Conclusion

This project helps in identifying different types of customers and understanding their behavior, which can be useful for targeted marketing strategies.
